Appendix A: Table of Shortcut Key Combinations

You can use the following keystrokes/key combinations to trigger RoboJournal program functionality from the main window:

F1 Display help file/documentation.
F2 Initiate a new connection.
F3* Terminate the current connection.
F4* Write a new entry.
F5* Modify the current (selected) entry.
F6* Manage tags for the current (selected) entry.
F7* Jump to the most recent entry.
F11* Expand/collapse chronological entry list.
Left Arrow* Display the entry immediately before the current (selected) entry.
Right Arrow* Display the entry immediately after the current (selected) entry.
DEL or DELETE* Delete the current (selected) entry.
CTRL + S* Export the current (selected) entry to a HTML document or a plain text file.
CTRL + P* Print the current (selected) entry.

* Requires an active connection.
